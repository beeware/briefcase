Briefcase
=========

.. image:: _static/logo.png
    :target: https://pybee.org/briefcase

Briefcase is a tool for converting a Python project into a standalone
native application. It supports producing binaries for:

* macOS,
* iOS,
* Android, and
* tvOS.
